TASK 1 : NUMBER GAME

Create a Java program that generates
a random number within a specified range
(for example, 1 to 100) and asks the
user to guess the number.

Key Requirements:
• Generate a random number using Java utilities
• Take user input for guesses
• Provide feedback if the guess is too high or too low
• Continue until the correct number is guessed

Optional Enhancements:
• Limit the number of attempts
• Allow multiple rounds
• Display the final score

Key Concepts:
Random class, loops, conditionals, input handling
